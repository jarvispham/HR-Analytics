📊 HR Workforce Analytics Dashboard
🎯 Project Overview
This project analyzes HR data to identify risks in employee retention and diversity. I built a dynamic dashboard that allows stakeholders to toggle between demographic categories and search for specific employee records.

🚀 Key Features
Dynamic Tab Navigation: Used Bookmark Navigators to separate "Problems" and "Key Findings."

Advanced Filtering: Implemented a multi-category search bar using Field Parameters.

Master Filter Summary: A custom DAX header that tracks active filters across the report.

💡 Key Insights
Retention Risk: Identified a 12.43% turnover rate, primarily affecting the Engineering and Finance departments.

Diversity Success: Achieved a near 50/50 gender split and robust ethnic distribution.

Succession Planning: Highlighted a "senior-heavy" workforce with less than 5% Gen Z representation.

🛠️ Tools Used
Power BI Desktop

Power Query (Data cleaning and transformation)

DAX (Custom measures and dynamic headers)
